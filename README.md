🔧 Installation & Setup

To run this project locally, follow these steps:

Clone the repository:

git clone https://github.com/SwatColombia/JR_PORTFOLIO.git
cd JR_PORTFOLIO

Install dependencies:

npm install

Run the development server:

npm run dev

Open in your browser:

http://localhost:3000
